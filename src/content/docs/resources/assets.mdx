Here are some assets that you can use to create content for SMC. All these images are `.png`, and pretty large to allow for resizing and editing. Whereever applicable the images also have transparent backgrounds. Vectors are useful for resizing as well, because they can be made as big as you need and then converted into any format. Feel free to create an issue [here](https://github.com/muhammadanwaystudent/smc-docs/issues) to request a certain type of asset.
